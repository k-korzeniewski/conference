
# How to run
### Type in command line 
```
  git clone https://github.com/k-korzeniewski/conference.git
  cd conference
  ./mvnw package
  java -jar target/*.jar
```
### Then open in browser 
```
  http://localhost:8080/
```
### H2 database console
```
    http://localhost:8080/h2-console
    Default login and password
```
